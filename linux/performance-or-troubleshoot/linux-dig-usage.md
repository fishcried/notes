# Dig命令使用

```
dit @8.8.8.8 www.baidu.com
```

**常用options**

| 参数    | 作用     | 使用场景   |
|---------|----------|------------|
| -t type | 指定类型 | 做类型查询 |

**重要**

| 参数         | 作用        | 使用场景 |
|--------------|-------------|----------|
| `+[no]tcp`   | 是否使用tcp | 不知道   |
| `+[no]trace` |             | 不知道   |
